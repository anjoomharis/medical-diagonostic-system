 Medical Diagnosis System

 Overview
This is a Python-based medical diagnosis system that predicts possible diseases based on user responses to symptom-related yes/no questions.

 Features
- Interactive yes/no symptom input
- Multiple disease detection
- Confidence percentage scoring
- Structured and modular code
- disclaimer included

Technologies Used
- Python
- Conditional logic
- Dictionary-based data handling

 How to Run
1. Download the project
2. Open terminal in project folder
3. Run:
   python medico.py

 Future Improvements
